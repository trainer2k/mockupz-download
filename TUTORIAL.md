# recordz tutorial

This guide shows how to capture app audio, trim the recording and export a WAV file.

## English

### 1. Complete the first-launch permissions

Before recording, follow [`INSTALLATION.md`](INSTALLATION.md) and enable recordz under **Privacy & Security → Screen & System Audio Recording**. Fully quit and reopen the app after granting permission.

### 2. Select the audio source

1. Open the source menu.
2. Choose the application whose audio you want to record.
3. Start playback in that application briefly and check that recordz shows audio activity.

Select the actual app that produces the sound—for example a browser, media player or communication app.

### 3. Start recording

1. Press the red record button.
2. recordz arms signal-start capture.
3. Start playback in the selected source app.
4. Recording begins when an audio signal is detected.
5. Press the red button again when the desired section is complete.

Signal-start helps avoid unnecessary silence at the beginning, but it is still worth checking the waveform before export.

### 4. Review the waveform

After recording, the waveform shows the captured audio.

- Drag the **left blue edge** to set the start.
- Drag the **right blue edge** to set the end.
- The displayed selection duration updates as you trim.

Leave a small amount of space around speech or music transients so the result does not sound abruptly cut.

### 5. Preview the selection

Press **Play** to hear only the selected section. Adjust the edges and preview again until the beginning and ending sound clean.

### 6. Export the WAV file

1. Press **Export**.
2. Choose or confirm the destination.
3. Open the exported WAV file and perform a final listen.

Use a descriptive filename that identifies the source and content. Make sure you have permission to record and use the audio.

### 7. Import an existing recording

recordz can also trim existing **WAV** and **MP3** files:

1. Import the audio file.
2. Wait for the waveform to appear.
3. Set the start and end edges.
4. Preview the selection.
5. Export the result as WAV.

### If no audio is captured

1. Confirm that recordz is enabled under **Screen & System Audio Recording**.
2. Quit recordz with **Command-Q** and reopen it.
3. Select the correct source application again.
4. Make sure that app is actively playing audio and is not muted.
5. Try a short test recording before capturing important material.

## Deutsch

### 1. Berechtigungen abschließen

Aktiviere recordz entsprechend [`INSTALLATION.md`](INSTALLATION.md) unter **Datenschutz & Sicherheit → Bildschirm- & Systemaudioaufnahme**. Beende die App danach vollständig und öffne sie erneut.

### 2. Audioquelle auswählen

1. Öffne das Quellenmenü.
2. Wähle die App aus, deren Ton aufgenommen werden soll.
3. Spiele kurz Ton in dieser App ab und prüfe, ob recordz ein Audiosignal anzeigt.

### 3. Aufnahme starten

1. Drücke den roten Aufnahmebutton.
2. recordz wartet auf das erste erkannte Audiosignal.
3. Starte die Wiedergabe in der ausgewählten Quell-App.
4. Beende die Aufnahme nach dem gewünschten Abschnitt.

### 4. Aufnahme schneiden

- Ziehe den **linken blauen Rand**, um den Start festzulegen.
- Ziehe den **rechten blauen Rand**, um das Ende festzulegen.
- Prüfe die angezeigte Dauer der Auswahl.

### 5. Auswahl anhören und exportieren

Nutze **Play**, um nur den markierten Bereich anzuhören. Passe die Auswahl bei Bedarf erneut an und klicke anschließend auf **Exportieren**, um eine WAV-Datei zu speichern.

### 6. Vorhandene Dateien bearbeiten

Du kannst auch WAV- und MP3-Dateien importieren, in der Wellenform schneiden, vorhören und als WAV exportieren.

Achte darauf, dass du die erforderlichen Rechte und Einwilligungen für aufgenommene oder importierte Inhalte besitzt.
